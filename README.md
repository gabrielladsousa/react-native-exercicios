# React Native Exercícios

Este repositório reúne os projetos desenvolvidos ao longo das aulas de React Native. Cada pasta contém um exercício diferente, criado para colocar em prática conceitos como componentes, gerenciamento de estado, hooks e interação com usuários.

## Projetos

| Projeto | Descrição |
|---------|-----------|
| 01-meu-perfil-profissional | Página simulando um perfil profissional, usando componentes e estilos. |
| 02-contador-de-pessoas-presentes-na-loja | Contador para acompanhar o número de pessoas em uma loja, exercitando **useState**, **props** e **button**. |
| 03-multiplicacao-de-2-numeros | Aplicativo simples para multiplicar dois números fornecidos pelo usuário. Prática de **inputs**, **cálculos**. |
| 04-alcool-ou-gasolina | Calculadora que indica se vale mais a pena abastecer com álcool ou gasolina, usando **condicionais** e **hooks**. |
| 05-calculo-do-imc | Calculadora de IMC (Índice de Massa Corporal) para prática de **inputs**, **cálculos** e **condicional**. |
| 06-jogo-do-num-aleatorio | Jogo que gera números aleatórios para o usuário adivinhar, trabalhando **useState** e **Math.random()**. |
| 07-anuncios-para-venda-de-produtos | App que exibe anúncios de produtos, com **scrollView** para rolagem horizontal. |
| 08-vagas-de-emprego-de-ti-com-scroll-view | Lista de vagas de emprego em TI usando **ScrollView** para rolagem vertical. |
| 09-vagas-de-emprego-de-ti-com-flat-list | Mesma lista de vagas, mas implementada com **FlatList**, prática de **renderização eficiente** de listas. |
| 10-formulario-de-abertura-de-conta-bancaria | Formulário completo para cadastro de conta, praticando **inputs**, **validações** e **state management**. |
